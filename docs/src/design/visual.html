§1Embassy Async Runtime Elimination
ELIMINATED COMPONENTS
Total eliminated ~61 KB
ARCHITECTURAL CHANGE
Tock/Embassy: Cooperative async executor. Each task is a state machine. Locals across .await captured in a static POOL.
pw-kernel: Synchronous event loop. Each process calls object_wait, processes one message, replies. No executor, no task pools.
HEAP: 16 KB → 4 KB
STACK IMPACT
Embassy: stack holds only one sync segment — async state is in POOL.

pw-kernel: full call chain runs on stack, but only one message in flight.

Net: −23.5 KB total (43.5 KB → 20 KB).
✦ Embassy removal frees ~47 KB BSS/heap and ~27 KB code that would be duplicated 4× (~108 KB total). Budget goes from "Conditional GO" to clear GO with 200+ KB headroom.
